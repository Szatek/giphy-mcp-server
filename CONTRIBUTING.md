# Contributing to Giphy MCP Server

We'd love your help to improve the Giphy MCP Server! By participating in this project, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).

## Getting Started

1.  **Fork** the repository and clone it locally.
2.  Install dependencies using `pip install -e ".[dev]"`.
3.  Create a branch for your feature or bugfix.

## Testing

Ensure your changes don't break existing functionality and add tests for any new behavior.
Run tests using:
```bash
pytest
```

## Creating a Pull Request

- Keep PRs focused on a single change.
- Ensure the CI workflows pass before asking for a review.
- Provide a clear description of what the PR solves.

Thanks for your contributions!
